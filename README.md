# Global Module

Exports global as a module


# Overview

This is a module which simply exports `global` and a CommonJS module.

This can actually be useful in some cases.


# Usage

CommonJS:

```js
const global = require('global-module');
```

ESM:

```js
import global from 'global-module';
```
